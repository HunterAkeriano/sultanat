# Sunset of the Crescent — кликер про Османскую империю (Godot 4)

2D-кликер/idle с пошаговым кризис-слоем по мотивам угасающей Османской империи.

> Стартовая эпоха — **1690 / Sultan Suleiman II / «The Decline»**, как на макетах.

## Как открыть и запустить

1. Установите **Godot 4.3** (или новее, ветка 4.x) — бесплатно: https://godotengine.org/download
2. Запустите Godot → **Import** → укажите файл `project.godot` в этой папке → **Import & Edit**.
3. Godot один раз импортирует ассеты (`.png` → `.import`). Дождитесь окончания.
4. Нажмите **Play** (F5). Игра запустится в портретном окне 430×932.

Запустить без редактора Godot нельзя — движок здесь не поставляется и проект
не собран в исполняемый файл. Откройте проект в Godot и нажмите Play.
