# Sunset of the Crescent — кликер про Османскую империю (Godot 4)

2D-кликер/idle с пошаговым кризис-слоем по мотивам угасающей Османской империи.
Собран **строго по ТЗ v1.1** и утверждённым макетам (stitch): экраны, дизайн-токены
(§21.2), палитра, шрифты, восковые печати-кнопки, тезхип-рамки, Летопись, Династия,
карточки событий («Имперский декрет»).

> Стартовая эпоха — **1690 / Sultan Suleiman II / «The Decline»**, как на макетах
> (открытый вопрос ТЗ B7 решён в пользу визуальной достоверности макетов).

## Как открыть и запустить

1. Установите **Godot 4.3** (или новее, ветка 4.x) — бесплатно: https://godotengine.org/download
2. Запустите Godot → **Import** → укажите файл `project.godot` в этой папке → **Import & Edit**.
3. Godot один раз импортирует ассеты (`.png` → `.import`). Дождитесь окончания.
4. Нажмите **Play** (F5). Игра запустится в портретном окне 430×932.

Запустить без редактора Godot нельзя — движок здесь не поставляется и проект
не собран в исполняемый файл. Откройте проект в Godot и нажмите Play.

## Что в игре (MVP-срез по ТЗ)

- **Трон** (§21.4.1) — Печать султана: каждый тап копит Хазну (чистый кликер, авто-дохода нет); под кнопкой — счётчик касаний и ценность одного касания; активная реформа «Низам-ı Джедид».
- **Эялеты** (§21.4.2) — карта-свиток, доход/сек, карточки провинций (Вложить / Подавить),
  рост трещин и потеря провинции при 100% (§11).
- **Институты** (§21.4.3) — фракции как апгрейды (Двор, Флот, Янычары [ВЫСОКИЙ РИСК], Улемы),
  ускорение реформы.
- **Летопись** (§21.4.6) — хроника решений на «шёлковой нити» с плашками эффектов.
- **Династия** (§21.4.5) — портрет правителя (свой на каждое правление, меняется при передаче Печати), Имперский престиж, ранги Дома, Legacy Decrees (за престиж), «Передать Печать наследнику» (новое правление с переносом мета-прогресса, §18, §21.0).
- **Кризис-слой:** Еда (мало → падает стабильность, растут бунты) и Оппозиция (каждое решение её двигает; при 100% — дворцовый переворот, смена правления). Чем выше оппозиция, тем чаще заговоры; чем меньше еды — тем чаще продовольственные бунты.
- **Событие/Имперский декрет** (§21.4.7) — модалка с тугрой, миниатюрой-поляроидом,
  восковыми кнопками-выборами, требованиями/ценой, затемнением фона.
- **Настройки** — переключение языка **RU/EN**, сброс прогресса, «Об игре».
- Автосейв в `user://sotc_save.json` (офлайн-дохода нет — это кликер). В любом событии гарантирован хотя бы один доступный (бесплатный) вариант — застрять нельзя.

## Шрифты (опционально, §21.2.2)

Положите три TTF в `assets/fonts/` (Bricolage Grotesque, Source Serif 4, Metrophobic) —
подробности в `assets/fonts/README.md`. Без них игра работает на встроенном шрифте Godot.

## Структура проекта

```
project.godot            — конфиг Godot (автозагрузки, портрет 430×932, главная сцена)
icon.svg                 — иконка (золотой полумесяц со звездой)
scenes/Main.tscn         — главная сцена (root Control со скриптом Main.gd)
scripts/
  Palette.gd             — дизайн-токены §21.2 (цвета, типографика, StyleBox-хелперы)
  Loc.gd                 — локализация RU/EN по ключам §29
  GameData.gd            — контент (провинции, институты, декреты, события) §24.2
  GameState.gd           — состояние + симуляция (тик §24.4, формулы §27, сейв §25.4)
  Main.gd                — корневой контроллер: HUD, 5 вкладок, навигация, оверлеи
  ui/Hud.gd              — шапка: султан/год/эпоха + 4 ресурс-чипа §21.4.1
  ui/ThroneView.gd       — Трон (Печать султана, реформа)
  ui/ProvincesView.gd    — Эялеты (карта, провинции)
  ui/InstitutionsView.gd — Институты
  ui/EventsView.gd       — Летопись
  ui/DynastyView.gd      — Династия
  ui/EventModal.gd       — модалка события
  ui/SettingsPanel.gd    — настройки + попап офлайн-дохода
assets/art/              — ассеты, вырезанные из утверждённых макетов
```

## Заметки по соответствию ТЗ

- Весь UI-текст — через ключи локализации (`Loc.t`), без хардкода (§29.2).
- Контент data-driven в `GameData.gd` (§24.2): события/провинции/институты — данные, не код.
- Палитра и формы строго из §21.2.1–21.2.4 (surface `#131313`, gold `#f2ca50`,
  emerald `#70db9d`, crimson, parchment `#f4ecd8`).
- Монетизация по §28: престиж только зарабатывается, без Pay-to-Win.

Это MVP-срез idle-слоя (§21.0). Полные 200 событий, 40 реформ, режим ПМВ, 20 концовок
и т.д. — следующие фазы дорожной карты (§33).
