# Шрифты (§21.2.2)

Чтобы интерфейс точно соответствовал утверждённой типографике ТЗ, положите сюда
три TTF-файла ровно с такими именами (всё подхватится автоматически при запуске):

| Файл                      | Семейство (Google Fonts) | Назначение                              |
|---------------------------|--------------------------|-----------------------------------------|
| `BricolageGrotesque.ttf`  | Bricolage Grotesque      | Заголовки экранов и карточек            |
| `SourceSerif4.ttf`        | Source Serif 4           | Тексты событий, цитаты-эпиграфы, числа  |
| `Metrophobic.ttf`         | Metrophobic              | Подписи вкладок, технические лейблы      |

Скачать бесплатно: https://fonts.google.com/ (поиск по названию семейства).

Если файлов нет — игра работает на встроенном шрифте Godot, просто без фирменной
типографики. Геймплей и вёрстка от этого не страдают.

> Подключение шрифтов реализовано в `scripts/Palette.gd` (`_load_fonts()`):
> файлы ищутся по путям `res://assets/fonts/<имя>.ttf`.
