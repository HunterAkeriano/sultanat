# Портреты жён (гарем)

Один портрет на жену (без возрастов). Конвенция: `wife_<N>.png`, аспект 3:4.

Кандидатки заданы в `scripts/GameData.gd → WIVES` (id → portrait):
- wife_1 → Гюльнуш
- wife_2 → Эмине
- wife_3 → Рабия
- wife_4 → Сафие

Если файла нет — в древе/событии рисуется медальон с инициалом (фолбэк).
Godot создаёт .import для новых .png при первом открытии проекта.
